    • Home page

    The "Home" page is the main page of our web site. From here you have easy access to see the main focus of the company, to learn more about us, as well as to get in touch with us. Furthemore, you have access to different projects of our architects in order to get in touch with there work. If you are a student or an architect, you are able to make your work public by simply sign in into our website.

    In the following section we will explain you how to do all of these.

    See all the projects

    To see all the projects, just click the button Pojects on the navigation bar.

    See all the articles

    If you are looking for articles, click the button Articles on the navigation bar.

    See all the events

    If you are looking for events, click the button Events on the navigation bar.

    Log in

    In order to add projects or articles, you will need to be logged in. To do that, all you need is to click the "Log in" button on the navigation bar.

    This acction will lead you to the "Login" page.

    Registration

    To create a profile, click on "Sign in" button.

    Doing this, a registration page will be opened.
